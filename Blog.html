Historia da Placa mãe

A placa mãe é um componente de hardware que liga todas as outras peças, fazendo a comunicação entre elas. A primeira placa mãe surgiu inicialmente em um computador da empresa IBM, no ano de 1982. O design das placas mãe continua basicamente o mesmo das primeiras, até os dias atuais. A placa da IBM assim como as sucessoras tem portas e slots para vários tipos de Hardwares, que são ligados nela para que a comunicação entre os componentes seja possível. HD, Memória, processador, Leitores Ópticos, Fontes, placas de vídeo, tudo é conectado à placa Mãe.

O que a placa mãe faz com as peças conectadas?

• Processador: A placa mãe transmite os “pedidos” de dados para a memória RAM e para o HD, e transfere estes dados para o processador

• Placa de vídeo: A placa mãe envia as informações e dados do Processador e HD para a placa de vídeo, e a placa de vídeo envia estes para o monitor.

• Memória RAM: A memória RAM sempre precisa de dados do HD, estes dados passam pela placa mãe para chegarem á memória.

• HD: A placa mãe pega as informações e dados do HD quando a memória RAM precisa.

• Leitor de CD/DVD: Quando o leitor interpreta os dados do CD ou DVD, ele envia estes dados para a memória RAM, que por sua vez manda para o processador. Estes dados são enviados para a placa de vídeo, que os transmite pelo monitor.

• Gravador de CD/DVD. Embora seja feito pelo mesmo equipamento, o sistema de gravação age diferente. O processador manda os dados á serem gravados para o gravador, que interpreta e grava os dados.

Tipos de placa mãe

• ATX: Um dos primeiros modelos de placa mãe, o significado de ATX é “Advanced” Technology Extended”. O seu tamanho é padrão para os primeiros modelos de computadores.

• EATX: “Extended ATX”, um modelo maior do que o padrão, por isso essa nomenclatura.

• Micro ATX: Essa placa mãe é menor que os dois modelos anteriores.

• Mini ITX: Mini Information Technology extended, esse modelo é menor que o micro ATX.
